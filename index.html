A Personal Access Token lets this tool write directly to your repo via the GitHub API — nothing else.
Need one? Create token ↗ (enable repo scope).

Before you proceed, understand:
  • Your token is used only for this single API call
  • It is never stored in localStorage or cookies
  • It is cleared from memory when you close this dialog
  • Communication goes directly to api.github.com

Deploy Your README

  1. Go to github.com/new
  2. Create a repo named exactly YOUR_USERNAME
  3. Check Add a README file → Create repository
  4. Click the pencil ✏️ icon to edit the file
  5. Select all (Ctrl+A), delete, paste your README
  6. Click Commit changes — done! 🎉
💡 The repo must be public and named exactly as your username.
